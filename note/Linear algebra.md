# 标量
定义：只有一个元素的张量
``` py
x=tf.constant(1.0)
```

# 向量
用一维张量表示向量，长度任意.**列向量**是向量的默认方向
``` py
x=tf.range(4)
x[3]    #取向量的任意元素，是标量
```

# 长度、维度、形状
求张量长度
``` py
len(x)
x.shape
```

# 矩阵
- 求转置
``` py
tf.transpose(A)
```
- 对称矩阵
- Hadamard积：相同形状的张量对应元素相乘，数学符号为（一个圈里面一个点）
``` py
A*B
```
